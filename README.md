# ☄️ Near-Earth Object (NEO) Hazard Prediction System

A Machine Learning-powered web application that predicts whether a Near-Earth Object (NEO) is hazardous or non-hazardous based on its astronomical characteristics.

## 📌 Project Overview

This project uses a trained Machine Learning model to classify Near-Earth Objects as:

- ⚠️ Hazardous
- ✅ Non-Hazardous

The application is built using Streamlit and allows users to enter asteroid parameters and receive instant predictions.

## 🚀 Features

- User-friendly Streamlit interface
- Real-time hazard prediction
- Machine Learning classification model
- Interactive input form
- Error handling for model and feature mismatches

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib
